# CodeAlpha Django E-Commerce Project

## Project Description
A Django-based E-Commerce website developed as part of the CodeAlpha Internship. 
This project allows users to view products, add items to cart, and manage products through Django Admin.

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite Database

## Features
- Product listing
- Product details
- Add to cart functionality
- User authentication
- Django Admin product management
- Responsive user interface

## Project Structure
- ecommerce - Django project folder
- store - Main application
- media - Product images
- manage.py - Django management file
